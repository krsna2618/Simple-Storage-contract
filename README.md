# Simple Storage Smart Contract

## Overview
A basic smart contract built using Solidity that allows storing and retrieving a number on the blockchain.

## Tech Stack
- Solidity
- Remix IDE
- Ethereum Virtual Machine (EVM)

## Features
- Store a number on-chain
- Retrieve stored value
- Gas-efficient contract

## How to Run
1. Open Remix IDE
2. Paste `SimpleStorage.sol`
3. Compile using Solidity compiler
4. Deploy using Injected Provider (MetaMask)

## Project Structure
- SimpleStorage.sol → Smart contract
- README.md → Documentation
